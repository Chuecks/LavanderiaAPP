# Assets

Esta carpeta debe contener los siguientes archivos de imagen para Expo:

## Archivos Requeridos

1. **icon.png** (1024x1024px)
   - Icono de la aplicación
   - Formato: PNG con fondo transparente o sólido

2. **splash.png** (1242x2436px recomendado)
   - Pantalla de inicio (splash screen)
   - Formato: PNG

3. **adaptive-icon.png** (1024x1024px)
   - Icono adaptativo para Android
   - Formato: PNG

4. **favicon.png** (48x48px)
   - Favicon para la versión web
   - Formato: PNG

## Nota Temporal

Si estos archivos no existen, Expo generará placeholders automáticamente. Puedes crear estos archivos más adelante o usar herramientas como:

- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator)
- [App Icon Generator](https://www.appicon.co/)

Para desarrollo inicial, puedes continuar sin estos archivos, pero son necesarios para producción.
