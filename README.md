# Proyecto Lavadero

Proyecto completo para la gestión de un negocio de lavandería.

## Estructura del Proyecto

```
Lavadero/
├── Front end/          # Aplicación móvil React Native con Expo
└── Back end/           # Backend del proyecto (pendiente)
```

## Frontend

La aplicación móvil está desarrollada con React Native y Expo. Para más información, consulta el [README del Frontend](Front%20end/README.md).

### Inicio Rápido

1. Navega a la carpeta Front end:
```bash
cd "Front end"
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor:
```bash
npm start
```
